Java面向对象总结
Posted on 2019-10-18   Updated on 2019-12-10

第二章 类和对象

1、面向对象的概念：

面向对象是把一个项目，一件事情，从头到尾一步一步执行，直到结束。

2、类的创建

File——>new –>class –>类名

3、创建对象

对象名.方法名(); 对象名.属性

4.构造方法：方法名和类名相同,没有返回值

5.构造方法重载

在同一个类中，方法名相同，参数个数，顺序，类型不相同的构造方法

6.构造方法的执行

7.static静态修饰符

静态修饰符可以修饰属性，方法，代码块
类名.属性 类名.方法名

8.java内存（栈和堆，static，方法区，gc（垃圾回收机制））

第三章 面向对象三大特征—封装

1.封装的概念

2.封装的步骤

(1).把属性私有化（private）
(2).对这些属性设置get和set方法
(3).调用处理这些方法

3.封装的作用

避免一些用一些不合乎常理的操作。

4.修饰符

public—>公共的：在一个项目中都可以访问，权限最高
private—>私有的：只能在本类中进行访问，权限最低
protected—>安全的，受保护的：同一个包能进行访问，继承中之类可以访问父类
default—>默认的：同一个包能访问

第四章 面向对象三大特征—继承

1.继承的概念

父亲儿子的继承关系。

2.继承的好处

(1)提高了代码的重用性
(2)提高代码的可扩展性

3.继承的关键字：extends

4.继承的语法：

5.继承的使用，在子类中：

可以直接调用父类的非私有属性和方法
可以用super.方法 super.属性

6.父类构造方法调用：super();

调用父类的构造方法必须在之类中调用
必须实在之类构造方法的第一行代码

7.重写@Override

必须继承
方法名一致，参数个数，顺序，类型全部一致

第八章 集合框架

1.集合框架的概念

集合框架是为表示和操作而规定的一种统一的标准体系结构。

2.List接口，Set接口，Map接口的区别

Set接口继承Collection接口，存储一组唯一(不允许重复)，无序的对象。
List接口继承Collection接口，存储一组不唯一(允许重复)，有序的对象。
Map接口存储一组成对的键-值对象，提供key(键)到value(键)的映射。Map中的key不要求有序，不允许重复。value同样不要求有序，但允许重复。

第九章 高级功能—–文件操作

1.文件概述

文件是具有内容和用途的，而文件夹是用来放置这些文件的。

2.什么是流

java流，是指一串流动的字符，是以先进先出的方式发送和接收数据的通道。

3.写文件

#####     (1)字节输出流写文件
int write(int c)—–写入一个字节数据
int write(byte[ ] buf)—–写入数组buf的所有字节
int write(byte[ ] b,int,off,int len)—–将字节数组从off位置开始，长度为len的字节数据输出大宋输出流
(2)字符输出流写文件
write(String str)—–将str字符串里包含的字符输出到指定的输出流中
write(String str,int off,int len)—–将str字符串里从off位置开始长度为len的字符输出到输出流中
void close()—–关闭输出流
void flush()—–刷新输出流

4.读文件

(1)字节输入流读文件
int read()—–从输入流中读取单个字符
int read(byte[ ] b)—–将数据读取到字节数组中
int read(byte[ ] b,int off,int len)—–从输入流中读取最多len长度的字符，保存到字符数组b中，保存的位置从off位置开始
void close( )—–关闭输入流
int available()—–返回输入流读取的估计字节数
(2)字符输入流读文件
int read()—–从输入流中读取单个字符
int read(byte[ ] b)—–从输入流中读取c.length长度的字符，保存到字符数组c中，返回实际读取的字符数
read(byte[ ] c,int off,int len)—–从输入流中读取最多len长度的字符，保存到字符数组c中，保存的位置从off位置开始，返回实际读取的字符数
void close()—–关闭流

第十章 高级功能—–多线程

1.线程的创建

(1)实现Runnable接口

步骤：
定义一个类实现Runable接口
实现接口中唯一的run()方法，该方法是线程线程开启时要执行的
在代码中声明一个线程实体Thread myThread=new Thread(download);//download 为实现Runable接口的类的实体对象
调用myThread.start()方法启动新创建的线程

(2)继承Thread创建线程

步骤：
创建类MyThread并继承Thread类
实现类MyThread的run()方法，该方法是线程开启时要执行的下载
生成MyThread类的对象，MyThread myThread=new MyThread();
调用myThread.start()方法启动新创建的线程
